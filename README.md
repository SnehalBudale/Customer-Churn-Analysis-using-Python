#  Customer Churn Analysis

##  Project Overview

This project analyzes customer churn data using Python and exploratory data analysis (EDA) to identify key factors influencing customer attrition. The analysis focuses on understanding customer behavior and deriving insights to improve retention strategies.

---

##  Objectives

* Identify key factors affecting customer churn
* Analyze impact of contract type, payment methods, and tenure
* Detect high-risk customer segments
* Provide actionable insights for customer retention

---

##  Dataset Description

The dataset includes customer-level information such as:

* Demographics (age group, senior citizen status)
* Contract type (monthly, yearly, long-term)
* Payment methods (electronic check, credit card, etc.)
* Tenure (customer duration)
* Service details and churn status

---

##  Tools & Technologies

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Jupyter Notebook

---

##  Analysis Performed

* Exploratory Data Analysis (EDA)
* Churn rate comparison across contract types
* Payment method vs churn analysis
* Tenure-based churn trend analysis
* Demographic-based churn analysis

---

##  Key Insights

* Month-to-month contract customers show highest churn (~42%)
* Electronic check users have highest churn (~45%)
* Customers with less than 1 year tenure show highest churn (~50%)
* Senior citizens and fiber users exhibit relatively higher churn

---

##  Visualizations

* Bar charts for churn comparison across categories
* Line graphs for tenure vs churn trends
* Distribution plots for customer segments

---

##  Recommendations

* Promote long-term contracts to improve retention
* Encourage secure and automated payment methods
* Focus on customer engagement during early tenure
* Design targeted strategies for high-risk segments
